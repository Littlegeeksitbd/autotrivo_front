# Autotrivo — React C

## 🚀 Setup & Run

### Requirements
- Node.js v18+ (download: https://nodejs.org)
- npm (comes with Node.js)

### Steps

```bash
# 1. Project folder এ যাও
cd autotirvo-clone

# 2. Dependencies install করো
npm install

# 3. Dev server চালু করো
npm run dev
```

Browser automatically open হবে: **http://localhost:3000**

---

### Build for Production

```bash
npm run build
npm run preview
```
